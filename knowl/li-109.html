However, given that the equation says that the function must be such that its derivative must be equal to the negative of itself it would be reasonable to guess that the function must be an exponential function. For \(y(x)=e^x\) we can see that
\begin{equation*} y'=e^x=y \end{equation*}
and so try instead \(y(x)=e^{-x}\text{.}\) Then
\begin{equation*} y'=-e^{-x}=-y \end{equation*}
and so we have found a (particular) solution.
