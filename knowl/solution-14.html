Solution 3.3.1.

  1. Since the argument of the log function has to be positive, the domain \(D\) is the set of points in the plane satisfying
    \begin{equation*} D=\{(x,y):y \lt x-1\} \end{equation*}
  1. \(\displaystyle f(3,1)=\ln(3-1-1)=\ln(1)=0\)
  2. \(f(\frac{1}{2},-\sqrt{5})=\ln(\frac{1}{2}+\sqrt{5}-1)=\ln(\sqrt{5}-\frac{1}{2})=0.5516\) (to 4.d.p.).
  3. Since \((1,3)\) is not in the domain of \(f\text{,}\) \(f(1,3)\) is not defined.
