Solution 12.4.1.

  1. What function has its second derivative zero (i.e. has zero curvature)? Clearly any linear function satisfies this property and so we guess that the general solution to this DE is
    \begin{equation*} y(x)=Ax+B, \; A,B\in \mathbb{R} \end{equation*}
  2. This DE can be rewritten as
    \begin{equation*} y''=y. \end{equation*}
    The solution, then, is any function that has its second derivative equal to the function itself. This is a property of any of the exponential functions
    \begin{equation*} y(x)=Ae^{x}. \end{equation*}
    It is also a property of the exponential functions
    \begin{equation*} y(x)=Be^{-x}. \end{equation*}
    Thus every function of the form
    \begin{equation*} y(x)=Ae^{x}+Be^{-x}, \; A,B\in \mathbb{R} \end{equation*}
    will be a solution to the DE.
